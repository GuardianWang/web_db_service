# Students RESTful API

## Usage

Start the MongoDB server.

```bash
npm install --dev
npm run devStart
```

Use [http://localhost:3000/studnets]().

To connect to MongoDB:

```
mongosh
connect("localhost/students")
>show dbs
>use students
>db.students.find()
```
